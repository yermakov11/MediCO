Крок 2: Налаштування серверної частини (Express.js)
2.1. Встановити Express.js, як залежність проекту за допомогою npm install express.+
2.2.Створити серверну структуру проекту, включаючи файли для маршрутів, моделей і контролерів.+
2.3. Створити базу даних (наприклад, використовуючи SQLite або MongoDB) та підключитися до неї.+
2.4. Реалізувати авторизацію/аутентифікацію для менеджера та пакувальника, можливо, 
    за допомогою Passport.js або іншої аутентифікаційної бібліотеки.+
2.5. --Створити маршрути для додавання, редагування, видалення замовлень та продуктів для менеджера.
2.6. Створити маршрути для перегляду списків продуктів та замовлень для пакувальника, 
а також для зміни статусу замовлення.
